## 提示：
### 你的`flatmap`模块应该完成如下功能：

```javascript
const flatmap = require('flatmap');

const arr1 = ['a', 'b'];

const duplicate = function(x){
    return [x, x];
};

flatmap(arr1, duplicate); // ['a', 'a', 'b', 'b']
```
