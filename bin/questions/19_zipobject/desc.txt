## 提示：
### 你的`zipobject`模块应该完成如下功能：

```javascript
var zipobject = require('zipobject');

zipobject(['fred', 'barney'], [30, 40]); // { 'fred': 30, 'barney': 40 }

zipobject(['Shanghai', 'Beijing'], [25000000, 30000000, 999]); // { 'Shanghai': 25000000, 'Beijing':  30000000}
```
