## 提示：
### 你的`once`模块应该完成如下功能：

```javascript
const once = require('once');

const getName = function(){
    console.log('Just say hi');
    return 'world';
};

const onceCall = once(getName);

onceCall(); //Just say hi
onceCall();
```
