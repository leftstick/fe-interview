* 在当前路径的`times`下，找到`index.js`，并补全实现代码

## 提示：
### 你的`times`模块应该完成如下功能：

```javascript
var times = require('times');

times(1, String); // ['0']

times(3, Boolean); // [false, true, true]

times(4, function(){ return 'Fuck'; }); // ['Fuck', 'Fuck', 'Fuck', 'Fuck']
```
